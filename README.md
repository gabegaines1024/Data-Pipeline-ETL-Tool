# Data Pipeline & ETL Tool

A production-ready ETL (Extract, Transform, Load) pipeline built with Python.

## Features

- **Multi-source extraction**: CSV, JSON, REST APIs
- **Smart caching**: LRU cache for API responses
- **Priority scheduling**: Job queue with priority levels
- **Data validation**: Comprehensive cleaning and validation
- **Monitoring**: Detailed logging and error tracking
- **Configurable**: YAML-based configuration

## Installation
```bash
# Clone repository
git clone <your-repo-url>
cd data-pipeline-etl

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Quick Start
```bash
# Run example pipeline
python main.py --config config/pipeline_config.yaml
```
